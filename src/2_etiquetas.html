2. Etiquetas para Textos

Las Etiquetas para textos te sirven para poder maquetar, maquetar quiere decir constriur la estructura de una pagina web

Tipos de etiquetas para Textos

h1 : sirve para poner le titulo de la pagina

h2 : utilizado para el sub titulo de la pagina

h3 : tambien para subtitulo pero cada vez de menor tamaño hasta llegar al h6

p: sirve para escribir parrafos


br: sirve para dar saltos de paginas

Lorem se utuliza para llenar un texto por defecto, al escribir Lorem te aparece el siguiente texto:


"Lorem ipsum dolor sit amet consectetur adipisicing elit. Excepturi, ipsam voluptate ad minus exercitationem facilis, aliquid temporibus vel at commodi consequuntur odio, velit ab cum dicta obcaecati culpa ducimus dignissimos?"

b: es la abrebiación de black y sirve para poner negrita una palabra o palabras

strong: visulamente tiene el mismo efecto que b, pero te sirve para posisionar tu pagina web, porque la palabra o palabras que esten dentro de "strong" son la que el buscador de google utilizara para ubicar o posisionar tu pagina, no es recomendable abuzar de esta, porque sino google te puede vetar.
