Tipos de listas

**Listas desordenadas

La lista desprdenada se hacen con el comando "ul" y dentro con el comando "li", ejemplo:

  • item #1
  • item #2
    • item #1
    • item #2
  • item #3
  • item #4

**Listas ordenadas

Es basicamente lo mismo que el anterior solo cambia el "ul" por "ol" y esto hace que la lista aparesca enumerada.Estas listas no suele ser la mas utilizada, pero es bueno tenerlo presente.

  1. item #1
  2. item #2
    1. item #1
    2. item #2
  3. item #3
  4. item #4
